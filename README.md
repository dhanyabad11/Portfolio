# Portfolio Website

A modern portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Modern, clean design inspired by academic portfolios
- Fully responsive layout
- Blog functionality with individual post pages
- TypeScript for type safety
- Tailwind CSS for styling
- Optimized for SEO

## Customization

- Update personal information in `src/app/page.tsx`
- Add your profile photo to `public/profile-photo.jpg`
- Modify blog posts in `src/app/blog/[slug]/page.tsx`
- Customize colors in `tailwind.config.ts`

## Build for Production

```bash
npm run build
npm start
```
